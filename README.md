# Exotic-Cats

A beautiful webpage showcasing the most rare exotic cats from around the world.

## About

This project features an educational and visually stunning webpage that highlights 9 of the rarest and most fascinating wild cat species on Earth, including:

- Sand Cat
- Black-Footed Cat
- Margay
- Pallas's Cat
- Fishing Cat
- Andean Mountain Cat
- Flat-Headed Cat
- Iberian Lynx
- Rusty-Spotted Cat

Each cat species includes detailed information about their habitat, unique characteristics, and conservation status.

## How to View

Simply open `index.html` in any modern web browser to view the webpage.

Alternatively, you can run a local web server:

```bash
# Using Python 3
python3 -m http.server 8080

# Then open http://localhost:8080 in your browser
```

- Responsive Design

## Conservation

These magnificent cats face numerous threats including habitat loss, human conflict, climate change, and poaching. Learn more about how you can help protect these incredible species by viewing the webpage.
